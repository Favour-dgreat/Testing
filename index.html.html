HTML - Hyper Text Markup Language

HTML is used for the basic structure of the website. The text, list items, tables, images,etc.

HTML is sometimes also refferred to as the skeleton of a website. Because it provides the bare content of the website, without any design or functionalities.
Over the years we have had so many versions of HTML, which includes:

  • HTML 3
  • HTML:XML
  • HTML:5

Note: HTML:5 is the most current version of HTML, though we expect HTML:6 to be released this year.

So the basic HTML Structure includes the Head and the Body.(Most people add the footer to this list but its not really proper, because the footer is supposedly part of what should be in between the body tags.)

Head

The Head section of a HTML file should contain the following:

  • The Title - This displays the title of the webpage.
  • meta Charset - This defines the current encoding that should be used in the website. And currently now it is UTF-8
  • meta name - This is used for the responsiveness of the website, and it is usually in this format: meta name ="viewport" content="width=device-width, initial-scale=1.0"

Body

The Body section of a HTML file is used to display the contents that should be displayed on the browser.
NOTE: Everything in between the body tag is what should be displayed on the website.
